# GeoLedger Documentation

This folder contains project documentation for GeoLedger.

## Files

| File                     | Purpose                                         |
| ------------------------ | ----------------------------------------------- |
| `architecture.md`        | System architecture, components, and data flow  |
| `api.md`                 | Backend API documentation                       |
| `smart-contracts.md`     | Smart contract design and blockchain logic      |
| `setup.md`               | Full local development setup                    |
| `development-roadmap.md` | Development phases, milestones, and commit plan |

## Documentation Rule

The root `README.md` should explain the project and how to run it.

Development history, phase planning, and internal build notes should stay inside this `docs/` folder.
